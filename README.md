# 物語メーカー Pro - GIGA Edition

「物語メーカー Pro」は、GIGAスクール構想の教育現場（主に小学校）向けに開発された、児童が楽しく作文や物語を書けるWebアプリケーションです。

バックエンドにGoogle Apps Script (GAS)、フロントエンドにReact + Tailwind CSSを採用しており、学校のGoogle Workspace環境で簡単にデプロイ・運用することができます。

## ✨ 主な機能

### 👧👦 児童向け機能（Student Mode）

*   **リッチなエディタ機能**: 自動保存（ブラウザ内）、文字数カウント機能を備えたシンプルな執筆環境。
    
*   **挿絵の挿入と編集**: Google Driveへ画像を自動アップロードし、サイズ調整や多彩な形（丸、星形、チケット型など）への切り抜きが可能。
    
*   **多彩なプレビューと印刷**:
    
    *   **原稿用紙モード**: 高学年（20×20）/ 低学年（15×16）の切り替え、禁則処理（追い出し・ぶら下がり）に対応した縦書き表示。
        
    *   **本モード**: 挿絵と文章が本のようにレイアウトされる見開き表示。
        
*   **先生からの添削確認**: 先生からの添削（赤線）とコメントを確認し、「修正完了」としてマークしていく機能。
    

### 🏫 先生向け機能（Teacher Mode）

*   **ダッシュボード**: 提出された作品の一覧表示、キーワード検索、状態（提出済・再提出・完了など）や日付によるフィルタリング。
    
*   **直感的な添削UI**: 原稿用紙プレビュー上で文字をドラッグして、直接コメント（赤線）を追加・削除可能。
    
*   **AI添削アシスト機能**: Gemini APIを活用し、児童の作文に対する誤字脱字の指摘やアドバイス（褒め言葉）をAIが自動生成。一括AI添削にも対応。
    
*   **一括操作**: 複数作品の一括完了処理、一括印刷機能。
    
*   **差分（Diff）表示**: 前回提出時からの変更箇所をハイライト表示。
    

### 🎨 みんなの作品だな（Gallery Mode）

*   「完了」または「提出済」となった作品をクラス全員で共有。
    
*   作品を読んで、お互いに感想（コメント）を送り合える交流機能。
    

## 🛠 技術スタック

*   **Frontend**: React (CDN), Tailwind CSS, Babel, Bootstrap Icons
    
*   **Backend**: Google Apps Script (GAS)
    
*   **Database / Storage**: Google Sheets (作文データ・コメント保存), Google Drive (挿絵画像保存)
    
*   **AI Integration**: Google Gemini API (`gemini-1.5-flash`)
    

## 🚀 導入手順（セットアップ）

このアプリはGoogle Apps Script (GAS) のWebアプリとして動作します。データベースとなるスプレッドシートや画像保存用フォルダは、**初回起動時に自動で生成（自己修復）される**ため、複雑な初期設定は不要です。

### 1. GASプロジェクトの作成

1.  [Google Apps Script](https://script.google.com/ "null") にアクセスし、「新しいプロジェクト」を作成します。
    
2.  プロジェクト名を「物語メーカー Pro」などに変更します。
    

### 2. コードの配置

1.  デフォルトで存在する `コード.gs` に、本リポジトリの `code.gs` の内容をコピー＆ペーストします。
    
2.  左側の「＋」ボタンから「HTML」を選択し、ファイル名を `index` にして作成します（`index.html` になります）。
    
3.  作成した `index.html` に、本リポジトリの `index.html` の内容をコピー＆ペーストします。
    

### 3. Webアプリとしてデプロイ

1.  右上の「デプロイ」>「新しいデプロイ」をクリックします。
    
2.  種類の選択（歯車アイコン）から「ウェブアプリ」を選択します。
    
3.  以下の設定でデプロイします：
    
    *   **説明**: （任意）
        
    *   **実行するユーザー**: `自分（あなたのGoogleアカウント）`
        
    *   **アクセスできるユーザー**: `全員` または `Google アカウントを持つ全員`（学校の要件に合わせてください）
        
4.  「デプロイ」をクリックします。
    
5.  初回は「アクセスを承認」の警告が出ます。「詳細を表示」>「物語メーカー Pro（安全ではないページ）に移動」から許可を行ってください。
    

### 4. 初期設定と動作確認

1.  発行された「ウェブアプリのURL」にアクセスし、アプリが表示されることを確認します。
    
2.  右上の「先生用」ボタンをクリックします。
    
3.  初回パスワードは **`admin`** です。
    
4.  ログイン後、右上の「設定（歯車アイコン）」をクリックし、以下の設定を必ず行ってください。
    
    *   **新しいパスワードの設定**（セキュリティのため必ず変更してください）
        
    *   **Gemini APIキーの設定**（AI添削機能を使用する場合。[Google AI Studio](https://aistudio.google.com/ "null") から取得したAPIキーを入力してください）
        

以上でセットアップは完了です。発行されたWebアプリのURLを児童に共有して使用を開始してください。

## 💡 ショートカットキー（先生用画面）

先生用画面では、キーボード操作で効率的に作業ができます（`?` キーで一覧表示）。

*   `↑` / `↓`: 児童の切り替え
    
*   `V`: プレビューの切り替え（原稿用紙 / 本 / 差分）
    
*   `S`: 表示中の作品を一括選択 / 解除
    
*   `U`: データの更新
    
*   `A`: AI添削の実行
    
*   `Shift + A`: 選択した作品の一括AI添削
    
*   `P`: 表示中作品の印刷
    
*   `Shift + P`: 選択した作品の一括印刷
    
*   `C`: 完了にする
    
*   `R`: 再提出にする
